# Pluto

**Pluto is the flexible, open-source image management service,
designed to store, organize, and serve images.
It provides a structured, reliable API for uploading, updating,
and retrieving media — making images easy to integrate into websites,
apps, and digital services.**


## Structured Image Management

Pluto allows images to be uploaded, annotated, and organized with
structured metadata, including:

- Alt text and captions for accessibility and search
- Focus points and cropping data for responsive display
- License and creator information for proper attribution

Images are stored efficiently and served via a robust API,
ensuring they are ready for web, mobile, or integration
into third-party services.

## Open Source and Licensing

Pluto is developed as an open-source project with a strong focus on transparency, collaboration, and long-term sustainability. It is released under a very permissive open license — allowing individuals, institutions, and developers to:

- **Reuse** the codebase for their own projects
- **Modify** and adapt the software to suit specific needs
- **Redistribute** it freely, with or without changes

This open model encourages integration into existing systems, custom extensions, and community-driven improvements.
